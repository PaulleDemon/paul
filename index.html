  • Python Designer

    Python designer is a visual scripting tool made to help python programmers easily create classes. A user is expected to add whats called as node from right-click context-menu, which then allows users to add variables and methods. Users can draw connection between nodes to depict inheritance. Once the users is satisfied with the design, user can click on generate to generate the python equivalent code.

  • Robotic arm controller

    This project was done during my 5th semester as final project for Robotics class.
    Robotic arm controller is a software that acts as an interface between, arduino that controls the robotic arm and you computer through bluetooth module. It provides a simple to use Graphical user interface, where users can add sequential instructions which will then be executed in the same order. Users can additionally save the instruction by clicking on save button and later load the instructions.

    Deploying web application to AWS is a tutorial that I authored as there was a need for good tutorial.
